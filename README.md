# MAIT-COVID19
R codes used in submitted article about MAIT and SARS-CoV-2 infection.

Doi: https://doi.org/10.1038/s41590-021-00870-z

## Informations 
This repo provided the following codes: 
- Circulatory plot.R
- Clinical Table.R
- Correlation and Corrplot.R
- FlowJoCyto Workflow.R
- PCA.R
